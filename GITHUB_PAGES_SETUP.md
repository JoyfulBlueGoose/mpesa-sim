# GitHub Pages Deployment Instructions

Your M-Pesa Simulator has been pushed to GitHub! Now follow these steps to enable GitHub Pages:

## Steps to Enable GitHub Pages:

1. Go to your repository: https://github.com/JoyfulBlueGoose/mpesa-sim

2. Click **Settings** (top menu)

3. Scroll down to **Pages** section (left sidebar)

4. Under "Build and deployment":
   - **Source**: Select "Deploy from a branch"
   - **Branch**: Select "main" 
   - **Folder**: Select "/ (root)"
   - Click **Save**

5. Wait 1-2 minutes for deployment

6. Your site will be live at: **https://joyfulbluegoose.github.io/mpesa-sim**

## Troubleshooting:

- **Site not showing after 5 minutes?** 
  - Check Settings > Pages to see deployment status
  
- **Want a custom domain?**
  - In Settings > Pages, add your custom domain
  
- **Need to update the app?**
  - Make changes locally
  - Run: `git add .`
  - Run: `git commit -m "your message"`
  - Run: `git push origin main`
  - Changes will deploy automatically!
